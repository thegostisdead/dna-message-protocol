# Graphical Protocol inspired of DNA

## Representation

We need 4 uniques proteins :

| Nucleotides Name | Nucleotides |   Base   | Color  |
|:----------------:|:-----------:|:--------:|:------:|
|     Adenine      |      A      |    0     |  red   |
|     Thymine      |      T      |    1     | green  |
|     Guanine      |      G      |    2     |  blue  |
|     Cytosine     |      C      |    3     | yellow |

StartCell value encoded :

- redundant = 200
- normal = 151

EndCell value encoded = 140
